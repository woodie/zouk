# zouk v1.3.0

## Testability: a `ScanHTTPClient` seam

`ScanClient`'s network calls now go through a `ScanHTTPClient` protocol instead of calling `URLSession` directly. `fetchScans()`, `cachedFile(for:in:)`, and `save(_:to:cacheDirectory:)` can all be exercised against a fake HTTP client in tests instead of hitting the real network.

## Bug fix: stale thumbnail cache

While testing the "no preview" placeholder by dropping a same-named file on the server, `cachedFile(for:in:)` was found to trust its on-disk cache forever, keyed purely by filename — silently serving the previous file's bytes (and thumbnail) under the new entry's name, with no error shown. It now checks the cached file's size against the server's reported size and re-downloads on a mismatch.

## Tests

`Tests/ZoukKitTests` is reorganized around each type's actual public methods rather than ad hoc scenarios, with real coverage for `cachedFile`/`save` in place of the old placeholders.

No other user-facing changes.
